# AI-Legal-Document-Risk-Assessment
An AI-powered system designed to simplify complex legal documents like contracts and privacy policies. Using NLP, it translates "legalese" into plain language, highlights critical clauses, and identifies potential risks. Empowering users without legal expertise to understand exactly what they are signing with clarity and confidence.
